# Prerequisites

1. [Nodejs](https://nodejs.org/)
2. [Npm](http://npmjs.com/)
3. [VSCode](https://code.visualstudio.com/)

# Install packages

Open a terminal in the root project folder and run the following command:
```shell
npm install
# or the shortcut
npm i
```

# Run the development server

Open a terminal in the root project folder and run the following command:
```shell
npm run start
```
